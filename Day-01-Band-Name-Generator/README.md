# 🎵 Band Name Generator

A simple Python program that generates a fun band name by combining the user's city name and pet name.

## 📌 Overview

The program asks the user for:

* The city they are from
* The name of their pet

It then combines both inputs to suggest a unique band name.

## ✨ Features

* Interactive command-line interface
* User input handling
* String concatenation
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run the Python file:

```bash
python main.py
```
