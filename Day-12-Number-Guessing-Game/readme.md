# 🎯 Day 12: Number Guessing Game

## 🎮 Project Overview

Built a command-line **Number Guessing Game** in Python where the computer randomly selects a number between **1 and 100**, and the player must guess it within a limited number of attempts.

The game includes difficulty levels, feedback after each guess, and win/loss conditions based on the player's remaining attempts.

---

## 🚀 Concepts Practiced

* Functions
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`while`)
* Variables and Scope
* Random Number Generation
* User Input Handling
* Game Logic Design

---

## ✨ Features

✅ Random number generation between 1 and 100

✅ Difficulty selection:

* Easy Mode → 10 attempts
* Hard Mode → 5 attempts

✅ Feedback for each guess:

* Too High
* Too Low

✅ Remaining attempts counter

✅ Win detection

✅ Loss detection when attempts reach zero

---

## 🧠 Challenges Faced

* Managing the attempts/lives system correctly
* Determining when the game should end
* Handling different outcomes (win or lose)
* Designing a simple but engaging game loop

---

## 💡 Key Learning

This project helped me understand how to manage program flow using loops and conditionals while building a complete interactive game. It also reinforced the importance of breaking a problem into smaller logical steps before implementing the solution.

As part of the lesson, I was introduced to the concepts of **scope** and **constants**, and I learned how they can improve code readability and maintainability in larger programs.
