# 🔧 Functions in Python

A beginner-friendly Python program demonstrating how to define and call a user-defined function. The project introduces one of the fundamental concepts of programming: **code reusability through functions**.

## 📌 Overview

The program defines a simple function named `my_function()` that prints two messages:

* Hello
* Bye

The function is then called, executing the statements inside it.

## ✨ Features

* User-defined function
* Function invocation
* Simple and readable structure
* Introduction to modular programming
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the Python file:

```bash
python main.py
```

## 💻 Sample Output

```text
Hello
Bye
```
