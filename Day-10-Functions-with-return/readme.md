# 🧮 Interactive Calculator

A command-line calculator built using Python that performs basic arithmetic operations and allows users to continue calculations using the previous result.

## 📌 Overview

The program enables the user to:

* Enter numbers
* Choose an arithmetic operation
* Calculate the result
* Continue calculating with the previous answer
* Restart the calculator whenever desired

The project uses Python functions and dictionaries to create a clean and modular design.

## ✨ Features

* Addition
* Subtraction
* Multiplication
* Division
* Continuous calculations
* Dynamic function mapping
* Interactive command-line interface
* ASCII art welcome screen

## 🛠️ Technologies Used

* Python 3

## ⚙️ How It Works

1. Accept the first number.
2. Display available operations.
3. Accept an operator and second number.
4. Perform the selected calculation.
5. Display the result.
6. Ask the user whether to continue with the current answer.
7. Repeat or restart based on the user's choice.
