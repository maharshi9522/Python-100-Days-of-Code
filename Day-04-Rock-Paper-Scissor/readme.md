# ✊✋✌️ Rock Paper Scissors Game

A simple command-line implementation of the classic **Rock, Paper, Scissors** game built using Python. The player competes against the computer, with the winner determined based on the traditional game rules.

## 📌 Overview

The program allows the user to select one of three options:

* Rock
* Paper
* Scissors

The computer randomly generates its choice, and the game determines the winner based on predefined conditions.

## ✨ Features

* Interactive command-line gameplay
* Random computer choice generation
* ASCII art representation for each move
* Win, lose, and draw conditions
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3
* `random` module

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the Python file:

```bash
python main.py
```

## 🎮 Game Rules

```text
Rock defeats Scissors
Scissors defeats Paper
Paper defeats Rock
Same choices result in a Draw
```
