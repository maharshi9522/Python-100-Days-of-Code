# 💰 Tip Calculator

A simple Python program that calculates the total bill amount including the tip and determines how much each person should pay after splitting the bill.

## 📌 Overview

The program asks the user to enter:

* Total bill amount
* Tip percentage
* Number of people sharing the bill

It then calculates the final amount and displays the amount each person needs to contribute.

## ✨ Features

* Accepts user input
* Calculates tip based on percentage
* Splits the bill equally
* Rounds the final amount to two decimal places
* Beginner-friendly command-line application

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run the program:

```bash
python main.py
```
